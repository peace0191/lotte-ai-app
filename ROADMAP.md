# 앱 로드맵 (브라우저 시연앱 v3 → Python/모바일 영업앱)

브라우저 시연앱(v3)은 로그인(OTP), 6개 단지 추천매물(18개), AI 저평가, 챗봇(FAQ), 교육환경 탭으로 구성되어 있습니다. 
이 구조를 Python(Streamlit)로 그대로 옮겨서 PyCharm/VSCode에서 실행하고, 모바일(크롬)에서 접속해 영업에 쓰도록 설계했습니다.

## UI 인덱스/페이지
0) Login: 휴대폰+이름 → 인증번호 받기 → 6자리 입력 → 로그인
1) Properties: 6개 단지 × 각 3개 카드(총 18), 선택 매물 세션 저장
2) Undervalued: 할인율+ML 기준 TOP 3, 선택 매물 세션 저장
3) Chatbot: 자연어 질문 + 민감키워드 감지 + 선택매물 컨텍스트 답변
4) Education: 대치동 학군/학원가 스토리 카드

## 모바일 영업 운영
- 노트북에서 실행 후, 같은 와이파이/핫스팟에서 휴대폰으로 `http://PC_IP:8501` 접속
- 외부 고객 공유용은 ngrok/Cloudflare Tunnel로 URL 발급
