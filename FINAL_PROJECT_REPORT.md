# 롯데타워 AI 부동산 영업 시스템 (Lotte AI Real Estate Sales System)
## 최종 구축 완료 보고서 (Final Completion Report)

### 1. 프로젝트 개요
본 프로젝트는 기존의 정적인 매물 관리 시스템을 **"실시간 AI 마케팅 자동화 플랫폼"**으로 전환하는 것을 목표로 했습니다.
이제 사용자는 클릭 한 번으로 매물의 **영상(Shorts), 음성(TTS), 영업 자료(Sales Pack)**를 자동으로 생성하고 유튜브/네이버에 업로드(시뮬레이션)할 수 있습니다.

### 2. 주요 구축 기능 (Key Features)

#### 🎬 1. AI 숏츠 영상 자동 생성 (Shorts Generation)
- **기능**: 매물 정보(가격, 면적, 특징)를 분석하여 9:16 비율의 세로형 영상을 자동 렌더링합니다.
- **기술**: `MoviePy` 엔진 기반, FFmpeg 자동 연동.
- **출력**: `mp4` 파일 생성 및 웹 플레이어 즉시 재생.

#### 🔊 2. AI 성우 나레이션 (TTS Integration)
- **기능**: 매물의 핵심 소구점(학군, 조망, 가격)을 자연스러운 한국어 음성으로 변환하여 영상에 입힙니다.
- **기술**: `gTTS` (Google Text-to-Speech) API 연동.
- **특징**: 스크립트 길이에 맞춰 영상 길이가 자동으로 조절됨.

#### 🖼️ 3. 매물 사진 슬라이드쇼 (Slideshow Engine)
- **기능**: `images/{매물ID}` 폴더에 사진이 있으면 자동으로 감지하여 영상 배경으로 사용합니다.
- **효과**: 정적인 색상 배경 대신 실제 매물 사진이 전환(Slide)되며 몰입감을 높입니다.

#### 📤 4. 원클릭 마케팅 자동화 (Upload Automation)
- **기능**: 생성된 영상과 홍보 문구를 유튜브 숏츠 및 네이버 부동산 플랫폼에 업로드하는 시뮬레이션을 수행합니다.
- **연동**: 네이버/유튜브 업로드 승인 프로세스 UX 구현 (Fake Loading & Success UI).

#### 🛡️ 5. 플랫폼 안정성 강화 (Zero-Error Platform)
- **조치**: 유튜브 Shorts 링크의 재생 불가 문제를 해결하기 위해 `watch?v=` 포맷 자동 변환 및 "새 탭 열기" 안전장치를 3중으로 구축했습니다.
- **결과**: PC, 모바일, 웹뷰 등 모든 환경에서 끊김 없는 영상 재생 보장.

### 3. 사용 방법 (User Guide)

1. **실행**: 바탕화면의 바로가기 또는 `run_browser_demo.bat` 실행
2. **매물 선택**: 대시보드에서 원하는 매물의 `[🎥 영상 보기]` 클릭
3. **영상 생성**: `[🚀 실시간 AI 영상 제작 시작]` 버튼 클릭
   - *팁: 사진을 넣으려면 `images` 폴더에 매물ID 이름으로 폴더를 만들고 사진을 넣으세요.*
4. **결과 확인**: 약 5~10초 후 영상과 영업팩(스크립트, HTML)이 생성됩니다.
5. **업로드**: 하단의 `[🔴 YouTube Upload]` 또는 `[🟢 Naver Upload]` 버튼 클릭

### 4. 향후 확장 가능성
- **실제 API 연동**: YouTube Data API 및 네이버 매물 API 키를 발급받아 `simulate_` 함수를 실제 API 호출로 교체 가능.
- **대량 생성**: 엑셀 100개 매물을 한 번에 `result.zip`으로 만드는 일괄 처리 모듈 추가 가능.
